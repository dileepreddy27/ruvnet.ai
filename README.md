# ruvnet.ai

`ruvnet.ai` is an AI infrastructure and multi-agent development platform for orchestration, automation, memory, and extensible developer tooling.

## Overview

This repository contains the core platform code for coordinating specialized AI agents across coding, planning, testing, documentation, and workflow execution. It is structured as a large TypeScript and Node.js codebase with support for MCP-style integrations, plugins, verification, and browser-facing experiences.

## Core Capabilities

- Multi-agent orchestration and workflow execution
- Memory and retrieval systems for persistent context
- Plugin and tool integration architecture
- Developer automation for testing, docs, and operations
- Verification, validation, and security-oriented utilities

## Repository Structure

- `ruflo/` runtime modules, app assets, and supporting platform code
- `v3/` next-generation packages, plugins, and orchestration components
- `plugins/` feature-specific extensions and integrations
- `docs/` product and technical documentation
- `tests/` automated test coverage
- `verification/` verification artifacts and validation records
